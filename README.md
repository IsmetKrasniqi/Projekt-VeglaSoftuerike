# Program për Llogaritjen e Pagës

Ky projekt është realizuar në C++ dhe llogarit pagën totale të një punëtori
bazuar në numrin e orëve të punës dhe pagën për orë.

Programi:
- lexon të dhënat nga përdoruesi
- kontrollon vlefshmërinë e input-it
- llogarit pagën bazë (deri në 40 orë)
- llogarit orët shtesë me koeficient 1.5
- shfaq pagën totale

Projekti është realizuar si punë ekipore duke përdorur GitHub dhe branches.
